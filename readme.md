## scripts
```javascript
 create [component] [name] [originalDirectory] [path]
```

| 字段      | 类型	     | 必填	     | 描述	     | 
| ---------- | :-----------:  | :-----------:  | :-----------:  | 
| type     | 'component' / 'page'      |  是     |  要复制的类型     | 
| name     | string      |  是     | 要命名的名称     | 
| originalDirectory | string      |  非     |   要复制的源文件路径     | 
| path     |  string      |  非     |  要复制的最后地址     |
